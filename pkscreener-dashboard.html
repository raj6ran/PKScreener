Add CI_PAT — Personal Access Token
Required for workflows to write scan results back to your repository.
1
Open GitHub token settings
Go to GitHub → Settings → Developer Settings → Personal access tokens → Tokens (classic)
https://github.com/settings/tokens
2
Generate new token (classic)
Click "Generate new token (classic)". Name it "PKScreener CI". Set expiry: 1 year.
3
Select scopes: repo + workflow
Check: repo (all checkboxes), workflow. Then click Generate token and copy it immediately.
4
Add CI_PAT and REPO_URL to secrets
Go to fork → Settings → Secrets → Actions → New repository secret
CI_PAT = <paste your token>
REPO_URL = https://github.com/raj6ran/PKScreener
