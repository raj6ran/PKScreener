ℹ️ Running PKScreener locally via Docker lets you see live scan results in your terminal without waiting for GitHub Actions market hours check.
Run with Docker (from PKScreener documentation)
Fastest way to see scan results locally on your Mac
STEP 01
Install Docker Desktop
Download from docs.docker.com/get-docker · Launch Docker Desktop · Login with Docker credentials
Download Docker ↗
STEP 02
Pull PKScreener image
Open Terminal on Mac (⌘+Space → Terminal) and run:
docker pull pkjmesra/pkscreener:latest
STEP 03
Run interactively
-i opens interactive mode · -t allocates a terminal
docker run -it pkjmesra/pkscreener:latest
STEP 04
Run specific scan
Use option strings to run specific scans directly:
docker run -it pkjmesra/pkscreener:latest -o "X:12:1"
