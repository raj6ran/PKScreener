Daily run order
Run in this sequence every trading day. Once all secrets are set the cron runs automatically.
1
Download stock data (w9)
Gets fresh OHLCV for all 2000+ NSE stocks. Takes ~3 min. Must be first.
Run w9 ↗
2
Warm up cache (w-data-warmup)
Pre-processes data so scans run 3x faster. Takes ~9s.
Run warmup ↗
3
Publish data (w-data-publisher)
Makes data available to all workers in parallel. Takes ~1 min.
Run publisher ↗
4
Nifty AI prediction (w4)
ML model predicts tomorrow's Nifty direction. Result sent to Telegram. Takes ~5 min.
Run w4 ↗
5
All production scans (w7)
Fires breakout, RSI, MACD, VCP scans across Nifty 50/100/500 in parallel. Results in Telegram in ~15 min.
Run w7 ↗
6
Check Telegram
Open @RAJESH_ALGO_bot — results arrive as messages with stock names, signals and indicators.
Open Telegram ↗
7
Add CI_PAT for auto-scheduling
Once CI_PAT secret is added, all steps above run automatically at market open (9:30am IST) and close (4pm IST) Monday–Friday.
https://github.com/settings/tokens → repo + workflow scopes → add as CI_PAT secret
